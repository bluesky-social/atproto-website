## 라벨에 대하여

atproto의 모더레이션은 다음을 포함한 여러 계층형 시스템으로 구성됩니다:

1. API에서 콘텐츠를 필터링하는 네트워크 차단
2. 모더레이션 서비스가 콘텐츠에 부여하는 라벨
3. 뮤트 및 차단과 같은 사용자 제어

클라이언트 애플리케이션을 개발하는 개발자는 라벨과 사용자 제어를 적용하는 방법을 이해해야 합니다.

**라벨**은 AT 생태계의 모든 계정이나 콘텐츠에 대한 메타데이터의 한 형태입니다. 라벨은 *모더레이션 서비스*에 의해 게시되며, 이 서비스는 애플리케이션에 하드코딩되어 있거나 사용자가 선택합니다. 라벨은 응답에서 `labels` 키 아래에 레코드에 첨부됩니다.

## **라벨 값**

라벨의 *값*이 그 동작을 결정합니다. 라벨 값의 예시로는 `porn`, `gore`, `spam` 등이 있습니다.

라벨 값은 문자열입니다. 현재 소문자 a-z 또는 대시 문자만 사용할 수 있습니다 `^[a-z-]+$`. 일부는 `!`로 시작하지만, 이는 전역 라벨 값에만 사용할 수 있습니다.

라벨 값은 정의에 따라 해석됩니다. 해당 정의에는 다음 속성이 포함됩니다:

- `blurs`: `content`, `media`, 또는 `none`
- `severity`: `alert`, `inform`, 또는 `none`
- `defaultSetting`: `hide`, `warn`, 또는 `ignore`
- `adultOnly`: 불리언 값

다른 정의 속성도 있지만 전역 라벨 값에서만 사용됩니다.

## **전역 라벨 값**

프로토콜에 의해 정의된 몇 가지 라벨 값이 있습니다:

- `!hide`: 클릭으로 통과할 수 없는 일반 경고를 콘텐츠에 표시하고 목록에서 콘텐츠를 필터링합니다. 사용자가 설정할 수 없습니다.
- `!warn`: 콘텐츠에 일반 경고를 표시하지만 클릭으로 통과할 수 있습니다. 사용자가 설정할 수 없습니다.
- `!no-unauthenticated`: 해당 라벨을 지원하는 애플리케이션에서 로그아웃된 사용자가 콘텐츠에 접근할 수 없게 합니다.
- `porn`: 이미지에 경고를 표시하며 사용자가 18세 이상이고 성인 콘텐츠를 활성화한 경우에만 클릭으로 통과할 수 있습니다.
- `sexual`: `porn`과 유사하게 동작하지만 덜 강렬한 성적 콘텐츠를 처리하기 위한 것입니다.
- `graphic-media`: `porn`과 유사하게 동작하지만 폭력/잔인한 콘텐츠를 위한 것입니다.
- `nudity`: 이미지에 경고를 표시하지만 18세 이상 제한이 없으며 기본적으로 무시됩니다.

전역 라벨 값이 존재하는 두 가지 이유가 있습니다.

첫 번째는 전역적으로 정의된 라벨 값만 자기 라벨로 사용할 수 있기 때문입니다(즉, Labeler가 아닌 사용자가 설정). porn, sexual, gore, nudity, !no-unauthenticated 라벨이 이러한 이유로 전역에 존재합니다.

두 번째는 "설정 불가" 및 "로그아웃된 사용자에게만 적용"과 같은 일부 특수 동작은 사용자 정의 라벨에 적용할 수 없기 때문입니다. !hide, !warn, !no-unauthenticated 라벨이 이러한 이유로 전역에 존재합니다.

## **사용자 정의 라벨 값**

Labeler는 자체 라벨 값을 정의할 수 있습니다. 모든 Labeler는 자신이 정의하는 라벨 값의 고유한 네임스페이스를 가집니다. 사용자 정의 정의는 `!`로 시작하는 것을 제외한 모든 전역 정의를 정의한 Labeler에 대해 재정의할 수 있습니다. `!`로 시작하는 것은 예약되어 있기 때문입니다.

두 가지 동작 속성(`blurs`와 `severity`)에 각각 세 가지 값이 있으므로, 사용자 정의 라벨 값에 대한 9가지 가능한 동작이 있습니다.

| **Blurs** | **Severity** | **설명** |
| --- | --- | --- |
| `content` | `alert` | 콘텐츠를 숨기고 조회 시 "위험" 경고 라벨을 표시 |
| `content` | `inform` | 콘텐츠를 숨기고 조회 시 "중립" 정보 라벨을 표시 |
| `content` | `none` | 콘텐츠를 숨김 |
| `media` | `alert` | 콘텐츠의 이미지를 숨기고 조회 시 "위험" 경고 라벨을 표시 |
| `media` | `inform` | 콘텐츠의 이미지를 숨기고 조회 시 "중립" 정보 라벨을 표시 |
| `media` | `none` | 콘텐츠의 이미지를 숨김 |
| `none` | `alert` | 콘텐츠에 "위험" 경고 라벨을 표시 |
| `none` | `inform` | 콘텐츠에 "중립" 정보 라벨을 표시 |
| `none` | `none` | 시각적 효과 없음 |

라벨에 사용할 수 있는 정의 예시:

- 괴롭힘: `blurs=content` + `severity=alert`
- 거미 경고: `blurs=media` + `severity=alert`
- 허위 정보: `blurs=none` + `severity=alert`
- 인증된 사용자: `blurs=none` + `severity=inform`
- 큐레이션 다운 조정: `blurs=none` + `severity=none`

`defaultSetting`은 사용자가 labeler를 처음 구독할 때 라벨이 어떻게 설정될지를 정합니다.

`adultOnly`는 성인 콘텐츠가 비활성화된 경우 라벨을 설정할 수 있는지 여부를 결정합니다.

## **라벨 설정**

사용자는 labeler의 각 라벨에 대해 숨기기, 경고, 또는 무시를 선택할 수 있습니다. 숨기기와 경고는 기본적으로 유사하지만, 숨기기는 라벨이 지정된 콘텐츠를 피드와 목록에서도 필터링합니다. 무시는 라벨을 그냥 무시합니다. 설정에서 성인 콘텐츠가 활성화되어 있지 않으면 재정의 없이 강제로 숨기기가 적용되어야 합니다.
