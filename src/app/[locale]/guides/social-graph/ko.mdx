## 소셜 그래프 조회

AT Protocol 위에 구축되는 새로운 애플리케이션은 기존 소셜 그래프 데이터를 활용할 수 있습니다. 예를 들어 `app.bsky.graph.getFollows` Lexicon RPC 엔드포인트를 통해 새로운 사용 사례를 만들고, 기존 사용자 기반으로 빠르게 시작할 수 있습니다. 이는 [소셜 파일시스템](https://overreacted.io/a-social-filesystem/)과 같은 atproto의 고유한 개념으로 이어집니다.

이 접근 방식의 또 다른 예시는 [Python으로 AT Protocol 탐색하기](https://davidgasquez.com/exploring-atproto-python/)를 참고하세요.

## 백링크

AT Protocol에서 소셜 그래프 데이터를 얻는 가장 강력한 방법 중 하나는 **백링크** 활용입니다. 백링크를 조회하면 특정 record 또는 identity를 가리키는 모든 record를 확인할 수 있습니다.

[Microcosm](https://constellation.microcosm.blue)은 백링크 작업을 위한 리소스 집합을 제공합니다. 모든 AT Protocol record는 공개적으로 접근 가능하고 `at://` URI로 주소 지정이 가능하기 때문에, 개별 URL에 중첩된 API 쿼리를 인코딩하는 것이 가능합니다. 예를 들어 Microcosm은 [이런 방식으로](https://constellation.microcosm.blue/xrpc/blue.microcosm.links.getBacklinks?subject=at%3A%2F%2Fdid%3Aplc%3Aa4pqq234yw7fqbddawjo7y35%2Fapp.bsky.feed.post%2F3m237ilwc372e&source=app.bsky.feed.like%3Asubject.uri&limit=16) 동작합니다.

백링크를 구현한 Claude code 플러그인은 [protopack](https://tangled.org/zzstoatzz.io/protopack.git)을 참고하세요.
