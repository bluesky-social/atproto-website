# Hello World

Get up and running with AT Protocol in just a few lines of code. This example fetches a public profile from the network â€” no authentication required!

## Install the SDK

```bash
npm install @atproto/api
```

## Fetch a Profile

Create a file called `hello.mjs`:

```javascript
import { AtpAgent } from '@atproto/api'

const agent = new AtpAgent({ service: 'https://public.api.bsky.app' })

const profile = await agent.getProfile({ actor: 'bsky.app' })

console.log(`Name: ${profile.data.displayName}`)
console.log(`Handle: ${profile.data.handle}`)
console.log(`Bio: ${profile.data.description}`)
```

Run it:

```bash
node hello.mjs
```

You should see output like:

```
Name: Bluesky
Handle: bsky.app
Bio: official Bluesky account (check usernameðŸ‘†)

Bugs, feature requests, feedback: support@bsky.app
```

That's it! You've just made your first AT Protocol API call.

## What's Happening

The `AtpAgent` connects to the public Bluesky API. The `getProfile` method fetches a user's profile by their handle or DID. Since this is a public read, no authentication is needed.
