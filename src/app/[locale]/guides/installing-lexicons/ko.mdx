## lex 사용하기

Lexicon 스키마는 타입과 유효성 검사를 포함한 코드 생성을 가능하게 합니다. 각 SDK에는 `lex` 도구가 포함되어 있어 각 AT 엔드포인트가 기대하는 파라미터를 알고 있는 타입 안전 클라이언트를 생성할 수 있습니다.

## TypeScript

먼저 `@atproto/lex` 패키지를 설치합니다:

```bash
npm install -g @atproto/lex
```

이를 통해 `lex` 명령어를 사용할 수 있으며, 로컬 프로젝트에 Lexicon을 설치하는 데 사용할 수 있습니다:

```bash
 lex install app.bsky.feed.post app.bsky.feed.like
```

이렇게 하면 다음이 생성됩니다:

- `lexicons.json` - 설치된 Lexicon과 버전(CID)을 추적하는 매니페스트
- `lexicons/` - Lexicon JSON 파일이 담긴 디렉토리

마지막으로 설치된 Lexicon에서 TypeScript 스키마를 생성합니다:

```bash
 lex build
```

이를 통해 `./src/lexicons`에 TypeScript 파일이 생성됩니다. 이제 코드에서 다음 함수를 사용할 수 있습니다:

```tsx
import { Client } from '@atproto/lex'
import * as app from './lexicons/app.js'

// Create an unauthenticated client instance
const client = new Client('https://public.api.bsky.app')

// Start making requests using generated schemas
const response = await client.call(app.bsky.actor.getProfile, {
  actor: 'pfrazee.com',
})
```

`lex` 사용에 대한 추가 안내는 [readme](https://github.com/bluesky-social/atproto/tree/main/packages/lex/lex#lexicon-schemas)를 참고하세요.
