Você deve ter instalado:
- Node.js 18+
- Go 1.25+
- O gerenciador de pacotes pnpm

[Léxicos](/guides/lexicon) definem o esquema para registros no Atproto. Para criar este tutorial, já publicamos um novo Lexicon "Statusphere" em nosso repositório, disponível em `xyz.statusphere.status`. Você pode usar o `lex` para baixar e construir esse Lexicon em seu projeto:

<S04 />

Se desejar, você pode visualizar o arquivo Lexicon baixado em `lib/lexicons/ xyz.statusphere.status.json`. A partir daqui, você pode executar `lex build` para gerar tipos TypeScript para todos os seus léxicos instalados:

<S05 />

<Note>
Você pode adicionar este comando ao `package.json`, se necessário, mas não o usaremos novamente neste tutorial, agora que nossos léxicos foram criados. </Note>
